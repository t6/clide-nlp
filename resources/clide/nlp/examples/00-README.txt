Welcome to clide-nlp!

This project contains some example texts that were used during the
development of clide-nlp.

cat.txt contains a sample of sentences with roughly the same meaning
that were used in the project iTac by the text equivalency group.

draw.txt contains an example text for the use case described in
Chapter 5 of the report.

----

Open a file and enable an annotation below the "nlp" user by clicking
on the little crossed out eyes.
Click somewhere in the text to realize the annotation for the chunk at
that point.
